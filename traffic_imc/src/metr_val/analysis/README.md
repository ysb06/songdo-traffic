# RNN 모델 성능 분석

## 1단계 예측 결과 생성 및 기본 분석 (prediction_comparison)

1. **모델 예측 수행** - 데이터로더의 모든 배치에 대해 모델 예측 실행
2. **예측 결과 저장** - HDF5와 pickle 형식으로 예측 결과 저장
3. **저장된 결과 로딩** - 저장된 예측 결과 파일 로딩 및 복원
4. **대화형 시각화 생성** - 실제값 vs 예측값 비교 차트 생성
    - 센서별 드롭다운 메뉴: 개별 센서 선택 및 전환
    - 월별 슬라이더: 시간 구간별 데이터 탐색
    - 실시간 호버 정보: 마우스 오버 시 정확한 값 표시
    - 확대/축소 기능: 특정 구간 상세 분석

## 2단계 에러 분석 (error_analysis)

1. **기본 회귀 메트릭 계산** - MAE, RMSE, MAPE, R² 등 핵심 성능 지표 산출
2. **센서별 에러 메트릭 분석** - 개별 센서의 예측 성능 평가 및 데이터 정제
3. **개별 에러 케이스 수집** - 모든 시점의 에러 계산 및 Top 10 최대 에러 추출
4. **종합 성능 평가** - 전체 성능 지표 및 센서별 순위 분석
5. **4차트 대시보드 시각화** - MAE 분포, 성능 산점도, Top 에러, 메트릭 상관관계 차트
