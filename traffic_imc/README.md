# Traffic-IMC 데이터셋 연구

이 폴더는 Traffic-IMC 데이터셋을 사용해 다양한 교통 예측 모델을 테스트하고 비교하기 위한 실험용 코드와 자료를 담고 있습니다.

## 주요 내용
- 데이터셋: Traffic-IMC 관련 파일과 전처리 스크립트
- 모델 실험: 여러 교통 예측 모델을 적용하고 성능을 평가하는 실험 스크립트

## 기본 개발 프로세스
- 외부 모델 참조와 구현체는 `../third_party` 폴더에 보관되어 있습니다.
- 필요하면 `models` 하위 폴더로 동일한 형태로 가져와(`import`) 사용하세요.
